1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. We should use constant for values that we do not want to change, and variables for values that we want to change. No, they cannot all be constants because some values need to be updated or modified during the process.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. You put the operations you want to be done first in parentheses. Operations in parentheses are performed before those outside parentheses.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. Using the toFixed() method, gives us the option to choose how many decimals we want.

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. One has the decimals rounded up while the other shows every decimals onward.

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Comparison operators return a boolean value, either TRUE or FALSE. We can check the result by using conditional statements like if-else to execute different code based on the boolean value.

  6. How does the Math.random() function work, and what can it be used for?

    7. Math.random() gives us a random number between 0 (inclusive) and 1 (exclusive). It can be used for generating random values.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. The modulus operator (%) returns the remainder of a division operation. It is useful for determining if a number is even or odd, or for wrapping around values within a certain range.
